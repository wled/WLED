# SkyStrip printable explainer

This folder generates a print-friendly HTML explainer from the single
source of truth in `../FAQ.md`.

## Build

From the repo root:

```sh
make -C usermods/usermod_v2_skystrip/explainer
```

Then open `usermods/usermod_v2_skystrip/explainer/explainer.html` and print
from your browser (or “Save as PDF”).
